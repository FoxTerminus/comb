# Benchmark Reports

Use these current files first:

- `final_report.md`: consolidated human-readable report.
- `current_benchmark_summary.csv`: benchmark-level scores and efficiency.
- `current_task_summary.csv`: task-level scores and efficiency.
- `current_failure_summary.csv`: collapse and packing diagnostics summary.

Older phase-by-phase reports were removed; the report directory keeps only the current consolidated results.
